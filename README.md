# Linear Regression From Scratch
This project implements the Linear Regression algorithm from the ground up using Pure Python. It focuses on the mathematical foundations of machine learning without relying on external libraries for the core logic.

### Mathematical Implementation
* Least Squares Method: Used to calculate the slope ($a_1$) and intercept ($a_0$) manually.
* Pure Logic: Calculated sums, squares, and products using basic loops to demonstrate deep understanding of the algorithm.
* Data Validation: Includes checks to ensure input consistency between $X$ and $Y$ values.

### Technologies Used
* Python
* Matplotlib (for visualization)
* Ast library (for input handling)
